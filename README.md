# 📝 Quiz App

An interactive Quiz Application built using **HTML, CSS, and JavaScript**.  
This project contains 10 multiple-choice questions with real-time score calculation and a clean user interface.

---

## 🚀 Features

- 📌 10 Multiple Choice Questions  
- 🎯 Instant Answer Selection  
- 📊 Final Score Display  
- 🔄 Restart Quiz Functionality  
- 💻 Simple & Responsive Design  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript
